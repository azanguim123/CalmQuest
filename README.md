# 🧠 CalmQuest – AI Mental Wellness Game

CalmQuest is a serious game that uses AI-powered facial emotion recognition to detect user stress levels and recommend personalized mini-games to improve emotional well-being.

## 🚀 Features

- 🎭 Real-time facial emotion detection
- 🧠 AI-based stress analysis
- 🎮 Adaptive mini-games (breathing, relaxation, focus)
- 📊 Progress tracking
- 🔒 Privacy-first (on-device processing)

## 🧩 How it works

1. Scan user's face
2. Detect emotional state (stress level)
3. Recommend personalized mini-game
4. Improve emotional well-being

## 🛠 Tech Stack

- Python
- PyTorch
- OpenCV
- Streamlit
